# f1-quiz

This is a Formula-1 quiz website made using HTML, CSS and JavaScript.

The quiz has 15 multiple-choice questions. Each question has four options and the user can select only one option at a time. The selected option gets highlighted. After selecting an answer, the user can click the “Next” button to go to the next question.

Only one question is shown on the screen at a time.

The score is calculated based on the number of correct answers. After the last question, the quiz ends and the final score is displayed on the screen.

The background image and styling are done using CSS, and all quiz logic such as loading questions, checking answers, and moving to the next question is done using JavaScript.
